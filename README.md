Dependencies
============
1) node/npm installed
2) git command line tools installed
Use
===
```
  git clone git@gitlab.cardinalsolutions.com:dabrams/angular-skeleton.git $PROJECT_NAME
  cd $PROJECT_NAME
  rm -rf .git
  git init
  git add .
  git commit -m "Initial commit"
  npm install
  gulp build
  gulp
  # goto http://localhost:8080/
```
